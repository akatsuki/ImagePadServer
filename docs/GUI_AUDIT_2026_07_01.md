# GUI Audit Notes - 2026-07-01

Scope: visual and interaction review only. No GUI behavior or styling changes were made in this pass, per the current-design-preservation rule.

## Candidates For A Future GUI Pass

- Tool install overlay: after failed tool installation, the overlay should make the next action clearer. Candidate copy: "再試行中" while automatic retries are active, and a concise manual recovery hint only after retries are exhausted.
- Pairing panel language: the pairing panel currently mixes English labels with the Japanese UI. Consider Japanese primary copy with the existing English terms retained only where product names require them.
- Queue tab label: "動画変換" is accurate, but the tab can also contain audio visualizer conversion work. Consider a broader label after confirming user expectations.
- Quit button prominence: the bottom quit action is intentionally available, but it is visually close to normal workflow controls. Consider making it slightly less dominant or adding spacing in a future design review.
- Progress accessibility: progress bars have labels, but dynamic percentage text is not always visible in the same region. Consider a compact text percentage for long-running conversions and installs.

## Deferred

- No color, spacing, layout, or typography changes were applied.
- No service routes, modes, tabs, or feature surfaces were added or removed.
- Follow-up backend fixes in this worktree also avoided GUI changes; the candidates above remain deferred for a dedicated UI pass.
